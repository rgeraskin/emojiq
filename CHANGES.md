# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-11-08

### Added
- Direct glyph matching search for emojis
- Help window with keyboard shortcuts
- Status hints for modifier key presses
- Multi-select functionality with Shift key support
- Functionality to increase emoji rank by 10

### Fixed
- Fixed truncated keywords in status bar

### Changed
- Improved increment_usage function to accept optional parameter

## [1.0.0] - 2025-10-20

### Added
- Settings window with customizable options
- Hotkey change settings
- App modes (emoji selection modes: paste only, copy only, paste and copy)
- Place under mouse cursor option
- Max top emojis setting
- UI scale factor adjustment and dynamic scaling for UI elements
- Reset emoji ranks button
- Emoji removal from most used list
- Dynamic window resizing and persistence of window size

### Fixed
- Fixed emoji alignment in layout

### Changed
- Optimized emoji write scheduling logic
- Improved logging and error handling

### Documentation
- Added visuals to README
- Added Homebrew installation instructions to README

## [0.1.0] - 2025-09-23

### Added
- Initial release
